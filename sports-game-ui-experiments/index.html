Что считается финальным результатом

Финальная верстка в этой сборке не пытается любой ценой сделать PNG поверх PNG. Значения, подписи, кнопки, шкалы, таблицы и CTA оставлены настоящими DOM-элементами, чтобы позже их можно было подключить к игровому состоянию. Raster используется только для иконок, гербов, портретов, tactical field layer и сложных декоративных glyphs.
