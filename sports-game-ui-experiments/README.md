# Sports Game UI Experiments GitHub Pages

Self-contained папка для публикации HTML-экспериментов на GitHub Pages.

Стартовая страница: `index.html`.

Основные разделы:

- `screen-01-match-hub-v2/original-mockup-blocks/block-map.html` - карта
  original/global/detail нарезки, live-DOM assembly map и clean live-DOM screen.
- `detail-blocks-live-dom-v4/compare-all.html` - сравнение всех 31 live-DOM
  блоков с original crop.
- `pipeline.html` - полное описание pipeline.
- `docs/live-dom-html-from-mockup-pipeline.md` - исходная markdown-версия pipeline.
