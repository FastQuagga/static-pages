# static-pages

Публичная репа для выкладки статичных HTML (демо, мокапы) через GitHub Pages.

## Разделы

- [`sports-game-visual-directions/`](sports-game-visual-directions/) —
  сравнение арт-направлений Match-экрана Sports Empire Idle
  (Broadcast / Retro Arcade / Neon Evolved), мокапы Claude и Codex, 2026-06-10.

## Как добавить раздел

1. Создать папку `<section-name>/` с `index.html` (самодостаточный HTML).
2. Добавить карточку-ссылку в корневой `index.html`.
3. Commit + push в `main` — Pages пересоберётся автоматически.
